IPL Data Analysis(Interactive Dashboard creation using Tableau)

Project Objective:
We all know the craze around IPL Cricket tournaments. All this time we have been discussing with our friend and colleagues analyzing which player did well which player did not do well, who would win and who would lose, which grounds are good for batting, and which are good for bowling, who would get man of the match and who would get man of the series etc.
While all these are based on every individual’s opinion and based on past performance and based on individual love for their favorite cricket player. Now we need to analyze using the data.

Content:
This data contains the match descriptions, results, winners, player of the matches, ball by ball dataset and much more.
This dataset consists of two separate CSV files: matches and deliveries. These files contain the information of each match summary and ball by ball details, respectively.
